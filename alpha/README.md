# BulnaAI - Alpha Version

このディレクトリには、React + Vite で構築されたα版のBulnaAIが含まれています。

## 概要

- **フレームワーク**: React 19 + TypeScript + Vite 6
- **デプロイ**: Vercel（mainブランチから自動デプロイ）
- **ステータス**: α版として維持中

## 開発コマンド

```bash
cd alpha
npm install
npm run dev     # 開発サーバー起動（ポート3000）
npm run build   # プロダクションビルド
npm run preview # ビルド結果のプレビュー
```

## 注意事項

- このディレクトリは**mainブランチ**でのみ使用されます
- **devブランチ**では、ルートディレクトリのNext.js版が開発中です
- α版の機能追加・バグ修正は、mainブランチから短命ブランチを切って実施してください
